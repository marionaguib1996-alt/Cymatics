Last Sync: Never
Push to Cloud sends everything currently in this browser up to Supabase — use it after Restore Academy to make a recovered backup durable, or any time you suspect the cloud has fallen out of sync. It never deletes anything from the cloud. Create Logins for Everyone sets up a login for every current student and teacher with an email set — their own email, and their ID as the password — safe to click more than once, already-created logins are updated in place. Admins are managed separately below.
Add Administrator
Creates a real login for a new administrator — they'll sign in with this email and the shared admin password.
Migrate Existing Staff Login
For an admin or teacher who already had a login before this scheme — resets their account to the current one (admin: shared admin password; teacher: their own ID as the password). Safe to run again if unsure whether it already happened.
Clean Up Redundant Multi-Course Logins
A student in two courses now logs in once and can switch between both — before this, each course had its own separate login. Run this once to remove any old, now-redundant logins left over from before. Safe to run any time; only touches accounts that are genuinely redundant.
Data Health Check
Scans for data that's fallen out of sync — a lesson whose student no longer exists, a student pointing at a deleted teacher, that kind of thing. Read-only: it only reports issues, it never changes or deletes anything on its own.
